# Ebay Command Line Selling Tool

## How to use:
	- Go to ebay's developer website
	- Sign up for a developer account
	- Create an API key
	- Copy app ID given to you
	- Run from the command line:
		- $python ebay-cli.py
	- Copy and paste ebay app id when prompted
